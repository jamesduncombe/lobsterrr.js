Javascript URI parser

THIS IS AN ALPHA RELEASE, AND IS NOT COMPLETE. DONT EXPECT IT ALL TO WORK YET!!

It's a faff parsing uris in javascript. This project is an attempt to put an end to this nastyness. Pass URI a uri and it will return a nicely organised javascript object for you to manipulate. Call URI.stringify on that, and it will return a uri string.

Usage

```:javascript // Parse a uri: var uri = URI.parse('http://www.nathansplace.co.uk:80/javascript/uri-js/docs/?array[]=one&array[]=two&hash[key]=value#something');

=> { protocol: 'http:', subdomain: 'www', host: 'nathansplace.co.uk', port: 80, path: ['javascript', 'uri-js', 'docs'], params: { array: ['one', 'two'], hash: { key: value } }, hash: 'something' }

// And put it back again... URI.stringify(uri);

=> 'http://www.nathansplace.co.uk:80/projects/javascript/uri-js/docs/?array[]=one&array[]=two&hash[key]=value#something' ```
